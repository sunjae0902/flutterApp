# image_font_add_app

앱에 이미지와 폰트를 추가해보는 프로젝트. 자주 사용되는 이미지나 폰트는 앱에 넣어 놓았다가 필요할 때마다 호출할 수 있다.<br>사용할 이미지나 폰트가 있다면 해당 앱 프로젝트 아래 image폴더에 추가하고, 플러터 앱의 버전 관리나 새로운 패키지, 이미지, 폰트 등을 추가하여 관리하는 pubspec.yaml 파일에 사용하려는 이미지의 정보를 추가한다.
